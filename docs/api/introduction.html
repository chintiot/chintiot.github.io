API 参考 ​

正泰智能云（Chint Smart Cloud）提供 RESTful API 和 MQTT 推送服务，方便开发者快速接入物联网平台。

服务概览 ​

服务说明
授权服务OAuth2 授权登录、访问令牌获取和刷新
用户服务用户注册、登录、信息管理等
产品服务产品品类查询、增删改查
设备服务设备绑定、查询、状态监控
消息服务设备配网、MQTT推送通知

通用说明 ​

  • 基础 URL: https://api.chintsmart.com
  • 请求格式: JSON
  • 字符编码: UTF-8
  • 公共参数参见 参数列表
  • 错误码参见 错误码字典

请求格式 ​

所有 POST 接口使用统一的请求报文格式：

json
{
  "message_id": "唯一消息ID",
  "user_id": "用户唯一ID",
  "params": {
    "access_token": "访问令牌",
    ...具体参数
  }
}

响应格式 ​

json
{
  "error_code": 200,
  "message_id": "唯一消息ID",
  "data": {}
}
