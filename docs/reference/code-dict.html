公共错误码 ​

错误码错误信息描述可尝试解决方法
200success成功N/A
400request error请求错误N/A
401request auth error请求认证错误检查 client_id/client_secret 是否已登录认证；检查配置是否正确；检查设备是否与当前用户绑定
403request forbidden请求被禁止检查 client_id/client_secret 配置；检查接口访问权限
404service not found服务未找到检查请求 method 和路径是否正确
429too many requests太多请求降低调用频次或稍后重试
460request parameter error请求参数错误检查上传的业务参数是否正确
500service error服务端错误联系服务人员
503service not available服务不可用联系服务人员
