物模型 ​

一般泛指 properties 对象，具体字段见下面的 JSON。

品类表 ​

category_id支持的品类品类说明
0开关switch
1插座outlet
2传感器sensor
3网关gateway
4指纹锁fingerprintlock
5灯light
6窗帘curtain
7摄像头camera
8万能遥控器telecontroller
9智能门控(门锁)smart-gating
10晾衣架hanger

0 开关 ​

请求示例 ​

json
{
    "top_nick_name": "xxx",
    "0": {
        "nick_name": "xxx",
        "power_state": true,
        "timing_timer": [
            {
                "enable": true,
                "repeat": "everyday",
                "start": {
                    "timer": "8:30",
                    "power_state": true
                },
                "end": {
                    "timer": "23:30",
                    "power_state": false
                }
            }
        ],
        "countdown_timer": [
            {
                "after_seconds": 3600,
                "power_state": true
            }
        ]
    },
    "1": {
        "nick_name": "xxx",
        "power_state": true,
        "countdown_timer": [
            {
                "remaining": 3600,
                "power_state": true
            }
        ]
    }
}

属性说明 ​

参数名类型说明
top_nick_namestring顶层开关的总昵称
nick_namestring开关的昵称
power_stateboolean开关的状态, 开: true, 关: false
timing_timerobject定时器
.enableboolean定时器是否开启, 开: true, 关: false
.repeatstring重复执行的方式: once=执行一次, everyday=每天执行, 1,2,3,4,5,6,7=自定义星期
.startobject定时开始时间及操作
.endobject定时结束时间及操作
countdown_timerobject倒计时定时器
.remaininginteger倒计时读秒, N秒后执行倒计时

TIP

power_state、timing_timer、countdown_timer 均为可选，一旦使用则其子项都为必选。

多控开关

0 和 1 指第一个和第二个开关，兼容了多控开关。若是单控开关则只需使用 0 即可。
